Walmart Weekly Sales Predictor — это простой проект по машинному обучению, в котором реализовано прогнозирование недельных продаж Walmart с использованием линейной регрессии. Работа разделена на три этапа: обучение модели, тестирование через терминал и создание веб-интерфейса на Streamlit.

Модель предсказывает Weekly_Sales на основе данных из датасета Walmart. Для обучения используются такие признаки как Temperature, Fuel_Price, CPI, Holiday_Flag, Month, Week, Year, Prev_Week_Sales, Prev_2_Week_Sales и Store. В проекте применяются Python, Pandas, Scikit-learn, Joblib, Streamlit и Matplotlib.

В первом этапе файл train.py загружает датасет, выполняет обработку данных, создаёт дополнительные признаки, обучает модель линейной регрессии, вычисляет метрики качества и сохраняет модель в файл model.pkl вместе со списком колонок model_columns.pkl. Запуск осуществляется командой python train.py, после чего выводятся MAE как средняя абсолютная ошибка, R2 как коэффициент детерминации и точность модели в процентах.

Во втором этапе файл test.py используется для проверки модели. Он загружает сохранённую модель, запрашивает у пользователя входные данные, выполняет предсказание и выводит результат в терминал. Запуск осуществляется через python test.py, после чего пользователь вводит значения признаков и получает прогноз продаж.

Третий этап реализован в файле app.py и представляет собой одностраничный веб-интерфейс. На странице отображается информация о модели, метрики качества, график зависимости фактических и предсказанных значений, часть исходного датасета, объяснение предобработки данных, форма для ввода параметров и кнопка для получения предсказания. Приложение запускается командой streamlit run app.py или python -m streamlit run app.py.

Для работы проекта необходимо установить библиотеки pandas, scikit-learn, matplotlib, streamlit и joblib. При использовании виртуального окружения установка выполняется через интерпретатор внутри .venv.

В процессе предобработки колонка Date была преобразована в отдельные признаки Month, Week и Year. Для каждого магазина добавлены значения продаж за предыдущую неделю и за две недели назад. Колонка Store была преобразована в набор бинарных признаков, так как линейная регрессия не должна воспринимать её как числовую величину. Weekly_Sales используется как целевая переменная и не входит во входные данные модели.

Датасет взят с Kaggle и содержит информацию о продажах Walmart. Проект демонстрирует полный цикл работы с моделью: от обучения до интеграции в веб-интерфейс.
